# 🤖 NeoDigital AI Toolkit

**Free open-source tools from [NeoDigital.tech](https://neodigital.tech) — Romania's #1 AI Agency**

A collection of production-ready AI utilities for developers and businesses.

---

## Tools Included

### 1. AI Customer Support Agent Starter
A ready-to-deploy AI support agent template using OpenAI + LangChain.
- Multi-language support (EN, RO, DE, FR, ES, EL, ZH)
- Escalation to human agents
- Customizable knowledge base

### 2. Lead Qualification Bot
AI-powered lead scoring and qualification pipeline.
- Integrates with any CRM via webhooks
- Scores leads 0-100 based on custom criteria
- Auto-routes high-value leads

### 3. AI Content Generator
Generate SEO-optimized blog posts and social media content.
- Powered by GPT-4
- Supports 7 languages
- Built-in SEO optimization

## Quick Start

```bash
git clone https://github.com/NeoDigitalAI/neodigital-ai-toolkit
cd neodigital-ai-toolkit
pip install -r requirements.txt
```
